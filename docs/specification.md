# プロセッサの仕様

このページでは，本実験で設計するプロセッサが満たすべき仕様について説明します．

## アドレスマップ
* $～は16進の数字を表します．
* アドレスはバイト単位です．
* 実装は 32bit データ幅の SystemVerilog コードに基づいています．
* データメモリは `$0000-$7fff`，I/O は `$8000` 以降です．

|アドレス|説明|
|--|--|
| $0000-$1fff | ユーザーが自由に使えるメモリ領域 |
| $2000-$3fff | ソートデータ領域 |
| $4000-$7fff | 追加のデータメモリ領域 |
| $8000  | ソート終了レジスタ（書き込み）．ソート終了時にプログラムから1を書き込む．ここに書き込むと，サイクル数のカウントが止まる． |
| $8004  | ソート数レジスタ（書き込み）．ソート終了時にプログラムからソート数を書き込む．|
| $8008  | デバッグ用LED（書き込み）．プログラムから書き込んだ値に応じて，ボード上のLEDを光らせる |
| $800C  | ソート開始レジスタ（書き込み）．ここに 1 を書くとサイクルカウントが開始される |
| $8010  | BTNU スイッチ（読み出し）．最下位 bit に現在の状態が入る |
| $8014  | BTND スイッチ（読み出し）．最下位 bit に現在の状態が入る |
| $8018  | BTNC スイッチ（読み出し）．最下位 bit に現在の状態が入る |
| $801C  | 現在のサイクル数（読み出し） |
| $8080-$809C | OLED 表示領域（Cycle 行） |
| $80A0-$80BC | OLED 表示領域（Size 行） |
| $80C0-$80DC | OLED 表示領域（Addr 行） |
| $80E0-$80FC | OLED 表示領域（Data 行） |

## プログラム

以下の1～4の処理を実装してください

1. BTNC（$8018）などの入力を読み出し，開始条件を満たすまで待機

	例：
	```
		sub $1, $1, $1       // $1 をゼロクリア
	WAIT:
		lw   $2, 0x8018($1)  // BTNC の読み出し
		andi $2, $2, 1       // 最下位ビットのみを取り出す
		beq  $2, $1, WAIT    // 0 だったら WAIT に戻る
	```

	注意：
	* BTNU, BTND, BTNC の各入力を使う場合，値の意味はボード入力の仕様に従います．
	* シミュレーションでは，開始条件を簡単にするために `$800C` に 1 を書いてサイクルカウントを開始しても構いません．

2. ソート実行
   * $2000～$3fff にあるデータを昇順にソートしてください．
   * 処理が終わった後，この領域に昇順にデータが格納されていればよいです．
   * （昇順：小さい値から順）
   * $0000～$1fff のメモリは自由に使って良いです．

3. ソート終了
   * ソート数レジスタ（$8004）に，ソートしたデータの個数を書き込んでください
   * ソート終了レジスタ（$8000）に，1を書き込んでください
   * ソート終了レジスタへの書き込みの例：

		```
		sub $1, $1, $1      // $1 をゼロクリア
		sub $2, $2, $2      // $2 をゼロクリア
		addi $1, $1, 1      // $1 を 1 に
		sw   $1, 0x8000($2) // $1 を $8000 にストア
		```

4. ソート結果の表示
   * OLED 表示領域 `$8080-$80FC` を使って，サイクル数，ソート数，現在見ているアドレス，そのデータを表示できるようにしてください
   * ボタン入力を使って表示対象を切り替えても構いません
   * 以下はC言語で書いた場合のこの処理のイメージ

		```
		volatile int* cycleRow = (int*)0x8080;
		volatile int* sizeRow  = (int*)0x80A0;
		volatile int* addrRow  = (int*)0x80C0;
		volatile int* dataRow  = (int*)0x80E0;
		short* address = (short*)0x2000;
			
		
		while(1){
			int dataValue = *address;
			int addrValue = (int)address;
			// 各行に 16 進 1 桁ずつ書き込んで表示するイメージ
			// 実際には下位 4 bit を順に `$8080-$80FC` に書き込む
			address++;
		}
		```
